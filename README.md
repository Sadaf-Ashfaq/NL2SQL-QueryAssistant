⚡ QueryCraft — NL2SQL Query Assistant

 Query any database in plain English. No SQL knowledge required.


Features:

1. Connect any database (SQL Server, PostgreSQL, MySQL, SQLite)

2. AI Agent — generates SQL, executes, and self-corrects errors automatically

3. Query History with search and filter

4. Schema & ERD Viewer — draggable diagram with relationships

5. Dangerous query guard (DELETE/DROP require confirmation)

6. Export results to CSV


🛠️ Tech Stack:
React + Vite + TailwindCSS | FastAPI | LLaMA 3.3-70B (Groq) | LangChain | SQL Server
